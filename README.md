# ai-tester
🧪 AI Playground — Test any AI model in one place. Paste your API key, pick a provider (OpenAI, Anthropic, Gemini, Groq, Mistral, DeepSeek &amp; more), tweak temperature, tokens &amp; parameters, and see real-time metrics like latency, speed &amp; cost. 🔒 Your API key stays 100% local in your browser — never sent to any server. Fully private &amp; secure.
